### Transfer files from client ssh to server sshd

#### transfer file

```scp /path/to/local/dir/file.txt mezcel@10.42.0.1:~/Downloads/```

#### transfer directory

```scp -r /path/to/local/dir user@remotehost:/path/to/remote/dir```
